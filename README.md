## Installation

1. Add dependency:
```xml
<dependency>
  <groupId>net.tassia</groupId>
  <artifactId>event-api</artifactId>
  <version>1.0.0-R0.1</version>
</dependency>
```

2. Install dependencies
```
$ mvn install
```
